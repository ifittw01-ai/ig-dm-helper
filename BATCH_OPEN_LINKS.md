# 📱 批量打开 Instagram 链接功能

## ✨ 功能说明

**自动在浏览器中一个一个打开 Instagram 用户主页！**

这个功能可以：
- ✅ 批量打开多个 IG 用户链接
- ✅ 自动在浏览器中显示用户主页
- ✅ 支持多种输入格式
- ✅ 自定义打开延迟
- ✅ 完美配合群发功能使用

---

## 🚀 快速开始

### 方法 1：双击运行（最简单）⭐

```
双击文件 → open-links.bat
```

### 方法 2：命令行

```bash
node open-ig-links.js
```

## 📋 支持的输入格式

| 格式 | 示例 | 说明 |
|------|------|------|
| 完整 URL | `https://www.instagram.com/username/` | ✅ 推荐 |
| 短 URL | `instagram.com/username` | ✅ 支持 |
| 用户名 | `username` | ✅ 支持 |

## 💡 使用技巧

### 技巧 1：准备链接列表

```
1. 在 Excel 或文本文件中准备链接
2. 复制所有链接
3. 粘贴到程序中
4. 批量打开
```

### 技巧 2：分批处理

```
如果有 50 个用户：
- 分成 5 批，每批 10 个
- 逐批打开和查看
- 避免浏览器卡顿
```

### 技巧 3：使用书签

```
对于常用的用户列表：
1. 批量打开后
2. 在浏览器中保存为书签文件夹
3. 下次可以直接从书签打开
```

## ❓ 常见问题

### Q: 最多可以打开多少个链接？

A: 理论上无限制，但建议：
- 每次 10-20 个（快速浏览）
- 每次 5-10 个（仔细查看）
- 避免浏览器卡顿

### Q: 浏览器没有打开怎么办？

A: 
1. 检查是否设置了默认浏览器
2. 尝试手动打开一个链接测试
3. 重启程序

### Q: 可以中途停止吗？

A: 可以！按 `Ctrl+C` 立即停止

### Q: 延迟可以设置为 0 吗？

A: 可以，但不推荐：
- 浏览器可能来不及加载
- 标签页太多会卡顿
- 建议至少 1-2 秒

### Q: 支持自动检测【訊息】按钮吗？

A: 目前不支持，需要您手动查看
- 自动检测需要登录和 API
- 手动查看更准确
- 未来版本可能支持
