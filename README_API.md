# 🚀 IG DM Helper - API 版本

## ✨ 完全繞過瀏覽器！

這是 Instagram DM Helper 的 **HTTP API 版本**，使用直接請求代替 Puppeteer 瀏覽器自動化。

---

## 🎯 快速開始（3 步驟）

### 1️⃣ 啟動應用

**Windows 雙擊啟動**:
```
START_API_VERSION.bat
```

**或命令行啟動**:
```bash
electron main-api.js
```

### 2️⃣ 登入 Instagram

在應用中輸入：
- Instagram 帳號
- 密碼
- 點擊「登入」

### 3️⃣ 開始使用

- 匯入帳號清單
- 編寫訊息
- 開始群發

**就這麼簡單！** 🎉

## 📦 核心功能

- ✅ **批量私訊** - 向多個帳號發送訊息

## 🔍 技術原理

### 瀏覽器版本 vs API 版本

```
🌐 瀏覽器版本:
用戶 → Puppeteer → Chromium → DOM 操作 → Instagram

⭐ API 版本:
用戶 → HTTP 請求 → Instagram API → 完成！
```

### 使用的技術

- **Electron** - 跨平台桌面應用
- **axios** - HTTP 請求庫
- **Instagram Private API** - 直接調用 API
- **SQLite** - 本地數據庫

## 🎓 使用指南

### 批量發送私訊

1. **準備帳號清單** (`accounts.txt`)
```
username1
username2
username3
```

2. **匯入清單**
   - 點擊「匯入 IG 帳號」
   - 選擇 TXT 文件

3. **編輯訊息**
   - 在訊息框輸入內容
   - 支援多行和 Emoji

4. **開始群發**
   - 點擊「開始群發」
   - 任務自動執行

### 匯出粉絲

1. **輸入帳號**
   - 輸入 Instagram 帳號名稱
   - 設定最多抓取數量

2. **開始匯出**
   - 點擊「開始匯出」
   - 等待完成

3. **保存文件**
   - 選擇保存位置
   - TXT 格式，每行一個用戶名

## ⚠️ 重要提醒

### 使用限制

| 動作 | 每小時 | 每天 |
|------|--------|------|
| 發送私訊 | 20-30 | 50-100 |
| 關注用戶 | 30-40 | 150-200 |
| 抓取粉絲 | 500 | 2000 |

### 風險警告

- ⚠️ 可能違反 Instagram 服務條款
- ⚠️ 帳號可能被限制或封禁
- ⚠️ 建議使用小號測試
- ⚠️ 控制使用頻率

### 安全建議

1. **使用小號**
   - 不要在主帳號上使用
   - 準備多個備用帳號

2. **控制頻率**
   - 每天不超過 50 條訊息
   - 分散在不同時間段

3. **內容質量**
   - 避免垃圾訊息
   - 提供真實價值
   - 個性化內容

4. **監控狀態**
   - 注意 Instagram 警告
   - 發現異常立即停止

## 🐛 常見問題

### Q: 登入失敗？

**A**: 
1. 檢查帳號密碼
2. 關閉二次驗證（2FA）
3. 先在手機 App 登入一次
4. 檢查網路連接

### Q: 發送失敗？

**A**:
1. 檢查對方是否封鎖您
2. 檢查帳號是否被限制
3. 降低發送頻率
4. 更換訊息內容

### Q: Session 過期？

**A**:
- Session 有效期 7-30 天
- 過期後重新登入即可
- 登入一次自動保存

### Q: 比瀏覽器版本安全嗎？

**A**:
- ✅ 更隱蔽（無瀏覽器窗口）
- ✅ 模擬 Instagram App
- ⚠️ 但仍有被偵測風險
- ⚠️ 建議控制使用頻率
