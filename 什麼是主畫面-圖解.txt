===============================================
   什麼是「主畫面」？圖解說明
===============================================

🎯 主畫面 = Electron 應用程序的主窗口
-----------------------------------------------

當你啟動程序後，會看到一個大窗口，
那就是「主畫面」！

🎬 操作步驟詳解
-----------------------------------------------

步驟 1：看到對話框
------------------
當群發失敗時，會彈出一個小窗口（對話框）：

   ┌─────────────────────────┐
   │ 🔐 請重新登入           │
   │                         │
   │ 請在主畫面中：          │  ← 這裡提到「主畫面」
   │ 1️⃣ 點擊【登入】按鈕    │
   │ 2️⃣ 輸入帳號密碼        │
   │ 3️⃣ 登入成功後點擊      │
   │    【確認完成】         │
   │                         │
   │ [✅確認完成] [❌取消]   │
   └─────────────────────────┘


步驟 2：找到「主畫面」
----------------------
「主畫面」= 對話框後面的大窗口

提示：
✓ 對話框後面那個大窗口
✓ 啟動程序時最先出現的窗口
✓ 有【初始化】【登入】【群發】按鈕的窗口
✓ 整個 Instagram DM Helper 的主窗口


步驟 3：在主畫面操作（不是對話框！）
------------------------------------

⚠️ 注意：是在「主畫面」（大窗口）操作，
不是在「對話框」（小窗口）操作！

1. 找到主畫面上的【登入】按鈕
   位置：通常在頂部或登入區域

2. 點擊主畫面上的【登入】按鈕
   （不是對話框上的按鈕！）

3. 在主畫面的輸入框中輸入：
   - 用戶名：你的 Instagram 帳號
   - 密碼：你的 Instagram 密碼

4. 再次點擊主畫面上的【登入】按鈕
   - 等待登入成功
   - 看到提示："歡迎，[用戶名]！"


步驟 4：回到對話框確認
----------------------
登入成功後：

1. 回到對話框（小窗口）
2. 點擊對話框上的【✅ 確認完成】按鈕
3. ✅ 程序會驗證登入狀態
4. ✅ 自動繼續發送


💡 記憶口訣
-----------------------------------------------

大窗口 = 主畫面 = 程序主窗口
小窗口 = 對話框 = 浮動提示窗口

操作流程：
對話框提示 → 去主畫面登入 → 回到對話框確認


🎯 關鍵理解
-----------------------------------------------

問：為什麼不能在對話框上登入？
答：對話框只是提示窗口，沒有登入功能。
    實際的登入功能在主畫面（主窗口）。

問：如果看不到主畫面怎麼辦？
答：對話框可能擋住了主畫面，
    你可以：
    1. 移動對話框到旁邊
    2. 或者先點【❌ 取消】關閉對話框
    3. 然後在主畫面登入
    4. 重新開始群發

問：主畫面長什麼樣子？
答：就是 Instagram DM Helper 的主窗口：
    - 有頂部菜單欄
    - 有【初始化】【登入】【獲取粉絲】等按鈕
    - 有登入區域（用戶名、密碼輸入框）
    - 有群發區域（匯入清單、訊息內容等）

❓ 常見問題
-----------------------------------------------

Q1：我看不到「主畫面」，只看到對話框？
A1：對話框可能擋住了主畫面，移動對話框
    或最小化它，就能看到後面的主畫面。

Q2：點了對話框上的按鈕沒反應？
A2：因為你需要先在「主畫面」登入，
    不是在對話框上登入！

Q3：在哪裡輸入帳號密碼？
A3：在主畫面（大窗口）的登入區域，
    有「用戶名」和「密碼」輸入框。

Q4：【確認完成】按鈕在哪裡？
A4：【確認完成】按鈕在對話框（小窗口）上，
    但你要先在主畫面完成登入，
    然後再回到對話框點擊【確認完成】。

Q5：我登入了但點【確認完成】說還沒登入？
A5：可能登入失敗了，檢查：
    - 用戶名和密碼是否正確
    - 是否看到「歡迎」提示
    - 等待幾秒確保登入完成


✅ 總結
-----------------------------------------------

記住這個關鍵點：

「主畫面」= Instagram DM Helper 的主窗口
         = 程序啟動後的大窗口
         = 有【初始化】【登入】【群發】按鈕的窗口
         = 對話框後面的大窗口

對話框只是浮動提示窗口，
真正的操作要在主畫面（主窗口）完成！


🚀 快速記憶
-----------------------------------------------

大窗口 = 主畫面 → 在這裡登入
小窗口 = 對話框 → 在這裡確認

流程：對話框提示 → 主畫面登入 → 對話框確認 → 完成！
